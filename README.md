# Vuno — AI Chatbot Frontend

Vuno is a modern AI chatbot frontend built with Next.js and Tailwind CSS. It provides a clean and responsive interface for interacting with an AI backend (RAG-based system) to query documents and get intelligent answers.

---

## Features

- Real-time AI chatbot interface
- Built with Next.js (App Router)
- Clean UI using Tailwind CSS
- Seamless backend API integration
- Context-aware AI responses (RAG)
- Fully responsive (mobile + desktop)
- Loading states & smooth UX

---

## Tech Stack

- **Framework:** Next.js 14+
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **State Management:** React Hooks
- **API Calls:** Fetch API
- **Image Handling:** next/image

---

## Installation

```bash
git clone https://github.com/sagardeepansh/vuno
cd vuno
npm install
```

---

## Run Locally

```bash
npm run dev
```

App will run on:
http://localhost:3000

---

## Core Components

- **ChatBox** → Handles chat state & layout  
- **InputBox** → Input + send logic
